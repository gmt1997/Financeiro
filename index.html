Fonte de dados

Local usa os dados históricos embutidos no app (e o que você adicionar fica salvo no navegador). Supabase persiste tudo em banco na nuvem.
Modo local ativo

Passos: 1) crie um projeto em supabase.com · 2) rode o schema.sql no SQL Editor · 3) cole URL e anon key acima e conecte · 4) clique em enviar histórico (uma vez só). Detalhes no README.

Importar planilha

Aceita .xlsx com as abas DADOS DRE, BTC e/ou Patrimônio (o formato da sua planilha), ou .csv exportado por este app. As duplicatas são ignoradas — pode re-subir o arquivo mestre sem medo.
